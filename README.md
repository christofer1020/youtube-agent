# YouTube Interest Profiler — Win95 Edition

An AI agent that scrapes your YouTube feed, analyses what it says about you
using **Groq (Llama 3)**, and presents the results as a fully interactive
**Windows 95 desktop** running in your browser — draggable windows, a Start
menu, a taskbar clock, and five live-switchable themes.

```
backend.py   →  opens Chrome, you log in, it scrapes your feed  →  calls Groq AI
             →  writes data.json  →  launches a local web server  →  opens the UI

index.html  ┐
style.css   ├─  vanilla JS/CSS frontend — reads data.json, renders a Win95 desktop
app.js      ┘   (no frameworks, no build step)
```

---

## What it does

1. Opens a real Chrome window and navigates to YouTube.
2. You log into your account, then press **Enter** in the terminal.
3. The agent scrolls your feed and collects video titles.
4. It sends those titles to **Groq's Llama 3** model, which builds a
   psychological interest profile — your dominant theme, watch style, top
   interests with confidence scores, and a "hidden passion" you might not have
   noticed yourself.
5. It writes `data.json`, starts a local web server, and opens the Win95
   desktop in your browser automatically.
6. You're asked: **"Is this accurate? [y/n]"** — answering **n** makes it
   scrape deeper (more of the feed, then your Watch History) and re-analyze
   until you're happy.

## Requirements

- Python 3.9+
- Google Chrome installed
- A free [Groq API key](https://console.groq.com) (takes ~1 minute to get)

## Setup

### 1. Install dependencies

```bash
pip install undetected-chromedriver selenium groq
```

### 2. Set your Groq API key

**Never paste the key directly into the code.** Use one of these instead:

**Option A — environment variable (recommended):**
```bash
# Mac / Linux
export GROQ_API_KEY="gsk_..."

# Windows (PowerShell)
$env:GROQ_API_KEY="gsk_..."
```

**Option B — `.env` file in the project folder:**
```
GROQ_API_KEY=gsk_...
```
Then load it before running:
```bash
# Mac / Linux
source .env   # or use python-dotenv

# Windows PowerShell
Get-Content .env | ForEach-Object { $k,$v = $_ -split '=',2; [System.Environment]::SetEnvironmentVariable($k,$v) }
```

> ⚠️ If you use a `.env` file, make sure `.env` is in your `.gitignore` so
> your key never ends up in a public repo.

## Running

### Full run (real scrape + AI analysis)

```bash
python backend.py
```

1. Chrome opens automatically — log into your YouTube account.
2. Press **Enter** in the terminal when you're ready.
3. The agent scrapes your feed, calls Groq, writes `data.json`, and opens
   **http://localhost:8000** in your browser.
4. Review the profile. Type **y** to confirm or **n** to dig deeper.
5. Press **Ctrl+C** when you're done.

### Demo run (no Chrome, no API key needed)

```bash
python backend.py --sample
```

Writes a realistic demo `data.json` and opens the UI — great for testing the
frontend or showing it off without touching YouTube or Groq.

## The Win95 desktop

Once the UI is open, you'll find these windows (open them from the desktop
icons or the Start menu):

| Window | What's inside |
|---|---|
| **My Computer** | Your dominant theme, watch style, video count, and when it was run |
| **Top Interests** | Your interest categories as cards with animated confidence bars |
| **Hidden Passion** | The subtle connection the AI spotted that you might not have noticed |
| **About You** | Full personality summary + three titles that represent you best |
| **Watch History** | Every video title that was scraped |
| **Display Properties** | Switch between 5 live themes (Win95, dark, synthwave, forest, dusk) |
| **Recycle Bin** | Easter egg |

Windows are draggable, minimisable, and closable. The taskbar shows open
windows and a live clock.
